### springstudy-seed-note 규칙
1. 주차별로 발표할 주제에 관한 내용을 `.md` 파일로 기록한다.
2. 파일명은 `[이름] 발표 주제` 형식으로 한다.
3. 동일한 형식으로 발표 **2일전까지** `pr`을 보낸다.
4. pr을 확인하고 **코멘트**를 단다.
5. 발표를 진행한 후 merge 한다.
